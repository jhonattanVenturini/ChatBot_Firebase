# ChatBot Firebase

## Descrição

Este é um projeto de chatbot simples desenvolvido em React. Ele permite que os usuários enviem mensagens e recebam respostas automáticas simuladas. O projeto inclui um botão para limpar o chat e utiliza estilização básica para uma interface limpa e funcional.

## Funcionalidades

- Enviar mensagens no chat.
- Respostas automáticas simuladas.
- Botão para limpar o chat.
- Interface responsiva e estilizada.

## Como Executar

1. Certifique-se de ter o Node.js instalado.
2. Clone este repositório:
   ```bash
   git clone https://github.com/jhonattanVenturini/ChatBot_Firebase.git
   ```
3. Navegue até o diretório do projeto:
   ```bash
   cd ChatBot_Firebase/chatbot
   ```
4. Instale as dependências:
   ```bash
   npm install
   ```
5. Inicie o servidor de desenvolvimento:
   ```bash
   npm start
   ```
6. Abra o navegador e acesse `http://localhost:3000`.

## Tecnologias Utilizadas

- React
- CSS

## Como Funciona

1. **Envio de Mensagens**: O usuário digita uma mensagem no campo de entrada e clica em "Enviar". A mensagem é exibida na janela do chat.
2. **Respostas Automáticas**: Após o envio de uma mensagem, o chatbot responde automaticamente com uma mensagem simulada.
3. **Limpar Chat**: O botão "Limpar" remove todas as mensagens do chat.

## Estilização

A estilização é feita utilizando CSS simples, com classes definidas para cada componente. O botão "Limpar" é destacado em vermelho para maior visibilidade.
